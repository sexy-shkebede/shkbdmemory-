# shkbdmemory
Легковесная и быстрая C# библиотека для прямого доступа к памяти процессов. Хорошая альтернатива kittymemory, поиск по паттернам (AOB) и никакой лишней шляпы

### Почему именно shkbdmemory?
* **Скорость**: Юзаем `unsafe` блоки и прямые вызовы WinAPI.
* **Умная запись**: Библиотека сама меняет права доступа к памяти (`VirtualProtectEx`), пишет данные и возвращает всё на место.
* **AOB Scanner**: Сама ищет сигнатуры с поддержкой масок (`??`). Нашел паттерн - получил адрес.
* **Чистый код**: Минимум комментариев, максимум логики.

### Как пользоваться:

1. Закинь `Memory.cs` в свой проект в vs
2. В настройках проекта (`.csproj`) обязательно включи: 
   `<AllowUnsafeBlocks>true</AllowUnsafeBlocks>`
3. Всё готово✌️
